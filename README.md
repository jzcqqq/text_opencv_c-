# OpenCV缺陷检测项目
## 开发环境
Visual Studio 2026 + OpenCV Windows库

## 使用说明
1. 使用者需自行本地下载、配置OpenCV依赖库；
2. 使用VS打开DE.sln解决方案编译运行；
3. sample文件夹为测试图片素材，需使用者自行准备，仓库未附带图片资源。
